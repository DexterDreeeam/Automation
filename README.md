# Win-Tester
Win-Tester is a UI automation test tool in **Windows** platform.  
Record your test case and rerun it immediately.  
  
## It support platform:
  - **Windows** only
  
## It support movements:
  - **UI Element Click**
  - **App Launch**
  - **Key Input**
